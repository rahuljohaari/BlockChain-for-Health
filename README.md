The 'BlockChain for Health' code has been written for the research paper title 'A Patient-Centric Consortium Blockchain Framework for Interoperable Electronic Health Records'. The Code has been written in Java Programming Language using ECLIPSE IDE.
For executing the code, download the .java files, create a new project in ECLIPSE IDE and add the two .java files.
The Dataset used in the simulation(Generation of Hash Values) has been downloaded from the URL:-
https://github.com/junioralive/Indian-Medicine-Dataset?utmsource
